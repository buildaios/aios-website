## Welcome to BuildAIOS!

BuildAIOS is an independent open-source organization building an AI-first operating system platform on top of Fedora Linux. 

### Key Features
- **Secure, Governed Executions:** Multi-tier sandboxing with Podman and SELinux.
- **Built-in MCP Gateway Core Proxy:** Unified routing and access enforcement for AI models and local tool execution.
- **Cognitive Control Plane:** Direct native system resource manipulation and secure runtime verification.
